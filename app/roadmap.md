# AI.Tube 项目战略规划蓝图 (2025版)

> 最后更新时间: 2025-12-24
> 当前阶段: MVP (最小可行性产品) 完成，迈向平台化

## 📍 1. 核心愿景
不仅仅做一个 AI 视频资源下载站，而是打造一个 **"AI 创作者的一站式成长与变现平台"**。
- **新人**：在这里学会第一句提示词。
- **创作者**：在这里找到工具、灵感和分镜。
- **专家**：在这里接到甲方的商业订单。

---

## 🗺️ 2. 功能版块规划

### ✅ 第一阶段：基石搭建 (已完成)
- [x] **用户系统**：注册/登录/个人中心/头像修改。
- [x] **积分体系**：每日签到/积分数据库/赚积分逻辑。
- [x] **资源交易**：视频详情页/智能下载/已购资产管理。
- [x] **视觉呈现**：赛博朋克风首页/大图推荐/呼吸动效。
- [x] **后台管理**：视频发布/分镜文件云存储/后台上传。

### 🚧 第二阶段：内容护城河 (进行中/即将开始)
**目标：通过高质量内容区分用户层级，沉淀核心资产。**

#### 2.1 👑 会员专区 (VIP Library)
- **定位**：高净值内容，通过“稀缺性”刺激付费/高频签到。
- **功能**：
  - 独立的黑金风格 UI 页面。
  - 仅限会员查看/下载的 4K 原片、工程文件 (Project File)。
  - 试看逻辑 (非会员只能看 30s 或模糊图)。

#### 2.2 🎓 新手教程专区 (Tutorial Zone) - **引流利器**
- **定位**：SEO 流量入口，降低新人门槛。
- **功能**：
  - 结构化课程：如《Sora 从入门到精通》、《Midjourney 分镜课》。
  - 图文 + 视频混排形式。
  - **关键逻辑**：教程免费看，但教程里用到的“提示词/分镜”需要跳转去资源区下载 (形成闭环)。

---

### 🛠️ 第三阶段：工具即服务 (Tools as a Service) - **核心竞争力**
**目标：从“看内容”转变为“用工具”，大幅提高用户粘性。**

#### 3.1 🧰 灵感工具库 (Creativity Toolkit)
这是一个工具集合页，包含实用的小工具：
1.  **提示词词典 (Prompt Library)**：
    - 像查字典一样查光影、镜头、风格词汇。
    - 点击即可复制，或者组合生成。
2.  **一键反推/拆解工具 (AI Analysis)**：
    - 用户上传视频 -> 接入 GPT-4V/Gemini API -> 自动分析分镜、景别、提示词。
3.  **分镜生成器 (Storyboard Gen)**：
    - 输入一句话 -> 自动生成 4-6 格分镜描述。

---

### 💰 第四阶段：商业变现闭环 (Business)
**目标：为成熟创作者提供价值，平台从“卖资源”转型为“抽佣/对接”。**

#### 4.1 🤝 商务需求大厅 (Demand Hall)
- **展示形式**：类似 Banner 或 任务列表。
- **内容**：
  - "某游戏公司寻 AI 视频制作，预算 5000元"
  - "某品牌寻产品展示视频"
- **交互**：
  - 初期：管理员后台发布，用户点击联系。
  - 后期：创作者在线投标/作品集投递。

---

## 📝 3. 技术栈升级预备
为了实现上述功能，后续需要引入的技术：
1.  **文章/CMS 系统**：用于支持教程专区的长图文排版 (可能需要 MDX 或简单的富文本存储)。
2.  **AI API 集成**：需要开通 OpenAI / Gemini / Claude API，用于实现“一键分析”和“生成工具”。
3.  **支付系统对接**：未来会员充值需要对接 Stripe 或 支付宝/微信支付。

---

## 📅 4. 近期执行优先级
1.  **VIP 会员专区** (完善内容分层)
2.  **新手教程专区** (引入流量)
3.  **提示词库** (工具库的第一步，技术成本相对低)
4.  **AI 高级工具** (需研发投入)
