# 🚀 AI.Tube 产品演进路线图 (2026 Roadmap)

> 最后更新：2025-12-25
> 当前阶段：Phase 2 - 商业化与工具化

## ✅ Phase 1: 核心架构与数据打通 (已完成)
- [x] **基础设施**
    - [x] Next.js 15 + Tailwind 搭建前端框架
    - [x] Supabase 数据库设计 (Videos, Users, Articles, Jobs, Banners)
    - [x] RLS (Row Level Security) 安全策略配置
- [x] **后台管理系统 (Admin)**
    - [x] 管理员登录门禁
    - [x] 视频管理 (含 B 站一键抓取、分镜上传)
    - [x] Banner 轮播图配置
    - [x] AI 学院文章发布与编辑
    - [x] 甲方需求发布与状态管理
- [x] **前台核心体验**
    - [x] 首页：动态轮播、热门/精选筛选、AI 悬浮助手
    - [x] 视频详情页：播放器、提示词复制、**收藏提示词**
    - [x] 合作中心：需求筛选、状态展示
    - [x] 个人中心：展示收藏的提示词、下载历史、积分状态

## 🚧 Phase 2: 商业化闭环 (进行中)
**目标**：跑通“流量 -> 注册 -> 付费”的商业模型。

- [ ] **支付系统集成**
    - [ ] 接入 Stripe 或 支付宝/微信支付 (通过第三方 API)
    - [ ] 增加 "充值积分" 页面
    - [ ] 增加 "开通 VIP" 订阅流程
- [ ] **用户激励体系**
    - [ ] 每日签到领积分功能
    - [ ] 邀请好友注册奖励机制
- [ ] **通知系统**
    - [ ] 站内信 (当有人回复评论、或需求有新进展时)

---

## 🛠️ Phase 3: AI 工具箱深度开发 (核心差异化)
**目标**：从“内容站”转型为“SaaS 工具站”。

- [ ] **CineFlow 分镜生成器 (MVP)**
    - [ ] 接入 LLM (如 DeepSeek/GPT) 优化提示词
    - [ ] 接入 MJ/SD 接口生成分镜图
    - [ ] 实现“剧本 -> 分镜 PDF”一键导出
- [ ] **Prompt 优化器**
    - [ ] 简单的“中文转英文长 Prompt”工具

## 📢 Phase 4: 运营与性能优化

    - [ ] 检查所有页面的手机端显示细节
